## git-bug board title edit

Edit a title of a board

```
git-bug board title edit [BUG_ID] [flags]
```

### Options

```
  -t, --title string      Provide a title to describe the board
      --non-interactive   Do not ask for user input
  -h, --help              help for edit
```
